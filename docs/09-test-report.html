テストレベル別の実施方針

  • 単体テスト: main.js / auth.js の入力処理・表示切替・ページネーションロジック

バグID発見日対象機能内容重大度対応状況
BUG-0012026-05-15セキュリティ（パスワード保護）db/README.md に「bcrypt推奨」と記載されている一方、実装（server.py）では salt + SHA-256 を使用しており、設計記述と実装内容に差異があった低修正済（仕様書を実装準拠に統一）
BUG-0022026-05-15セッション管理ログイン時のトークンを localStorage と Cookie の両方で保持する実装となっており、運用ルールが曖昧な場合に管理が複雑化する可能性がある低保留（次版で保持方式を一本化予定）
BUG-0032026-05-15画面表示（詳細画面）cafe-detail.js の詳細データがフロント側の静的定義となっており、DB更新内容と自動同期されない中対応中（次版で詳細取得APIを追加予定）

バグ管理一覧 補足

本システムでは、開発中に発見した不具合を一覧化し、「原因」「影響範囲」「対応状況」を管理している。特に、認証・セッション管理・データ同期に関する機能は、システム全体へ影響するため重点的に確認を行った。
